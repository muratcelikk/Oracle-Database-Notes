-- Sýk kullanýlan formatlarý aþaðýda açýkladým.

%a     Varolan veritabaný aktivasyon idsi 
%A     0 ile tamamlanmýþ aktivasyon idsi 
%c     Maksimum deðeri 256 olan backup piece kopya numarasý 
%d     Veritabaný adý 
%D     DD formatýnda gün bilgisi 
%e     Arþiv log dosyasý sýra numarasý 
%f     Absolute dosya numarasý 
%F     DBID, gün ay, yýl ve benzersiz bir sýra numarasý içerir. 
%h     Arþiv redo log thread numarasý 
%I     DBID 
%M     MM formatýnda ay bilgisi 
%n     Veritabaný adý 
%N     Tablespace adý. Veri dosyasý ve image copy yedekle set edilebilir. 
%p     Backup set içindeki backup piece numarasýdýr 
%r     Resetlogs ID 
%s     Backup set numarasý 
%S     0 ile tamamlanmýþ sýra numarasý 
%t     Backup set zaman bilgisidir. Saniye bilgiside içeren 4 byte lýk bir deðerdir. 
%T     Gregorian takviminde YYYYMMDD formatýnda olan yýl, ay ve gün bilgisidir 
%u     Backup set veya image copy yedeðin numarasý ve oluþturulma zamanýný içeren 
       8 karakterlik bir bilgidir. 
%U     Sistem tarafýndan üretilen benzersiz bir numaradýr. 
%Y     YYYY formatýndaki yýl bilgisidir
